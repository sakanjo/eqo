## 📦 Install

```bash
cargo install eqo
```

Or you can download it from the [releases page](https://github.com/sakanjo/eqo/releases).

## 🦄 Usage

Run the server in the background, or as a service

```bash
eqo server -q
```

Start listening

```bash
eqo listen 'date'
```

Trigger the command

```bash
eqo run
eqo run
# ...
```
